Проект представляет из себя единый интерфейс для работы с заявками с различных проектов с разными типами анкет
Схема базы данных - иерархическая (Предполагается / MongoDB) и строится на основании загружаемых данных через API

API

Стороние проекты могут выгружать данные при помощи 3-ёх методов, реализованных на Process Request



add_orders

Принимает массив заявок


add_objects

Принимает массив объектов


delete_objects

Принимает массив id-шников для удаления
